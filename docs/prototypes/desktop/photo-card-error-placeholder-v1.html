设计说明

  • 占位图使用 bg-page + text-tertiary，与项目 tailwind.config.js 定义的主题一致。
  • Grid 模式下占位图使用 aspect-square，与正常缩略图尺寸完全一致。
  • Masonry 模式下占位图使用 h-auto min-h-[160px]，给卡片确定高度，避免破坏瀑布流布局。
  • 选中状态使用蓝色边框 + 左上角勾选图标，与正常卡片一致。
  • hover 效果保留 hover:scale-105 和阴影，保持交互一致性。
  • 不引入任何外部图片资源，仅用 SVG + Tailwind CSS。
