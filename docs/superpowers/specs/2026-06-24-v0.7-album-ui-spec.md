# v0.7 相册管理 UI 规格说明

> **状态**: Draft
> **创建日期**: 2026-06-24
> **关联原型**: [album-management-v1-light.html](../../prototypes/desktop/album-management-v1-light.html)
> **关联设计稿**: [v0.7-quality-polish-design](2026-06-23-v0.6-quality-polish-design.md) 的下一版本

本文档详细定义 v0.7 相册管理功能在 5 个 sidebar sector 中的所有支持操作、功能矩阵、上下文菜单和拖拽行为。
作为前端实现、后端 API、QA 测试的统一参考。

---

## 1. Sector 总览

v0.7 在 v1 的 sidebar 基础上新增了 5 个 sector，**统一 3 层布局**：

```
<sector>
  ┌──────────────────────────────────────┐
  │ <toolbar>     标题 + 通用操作 + 视图操作 │  ← 第 1 层
  │ <view-tabs>   视图专属 tabs            │  ← 第 2 层
  │ <content>     photo-grid / albums-grid │  ← 第 3 层
  │               / timeline-view         │
  └──────────────────────────────────────┘
```

| 顺序 | Sector | 触发 | 数据源 | 主内容 |
|------|--------|------|--------|--------|
| 1 | **⭐ 收藏** | 点击 sidebar | `photo.is_favorite=1` | 照片网格 |
| 2 | **🕐 时间线** | 点击 sidebar | 全部照片 | 按月分组瀑布流 |
| 3 | **📅 按年份浏览** | sidebar disclosure | `path=YYYY/YYYY-MM` | 照片网格 |
| 4 | **🖼 相册集** | 点击 sidebar | 全部相册 | 相册大卡片 |
| 5 | **📁 文件夹** | 点击 sidebar 任意子项 | 根目录 | 照片网格 |

## 2. Sector 1：⭐ 收藏

### 2.1 入口
- Sidebar: `⭐ 我的收藏 (156)` 整行可点击
- 工具栏标题：`我的收藏` + 数量徽章 `156 张`
- URL 状态：`/favorites`

### 2.2 Toolbar 操作（从左到右）

| 位置 | 按钮 | 图标 | 功能 | 触发 | API |
|------|------|------|------|------|-----|
| 左侧 | 标题 `我的收藏` | — | 标识当前视图 | — | — |
| 左侧 | 数量 `156 张` | — | 实时显示当前 sector 照片总数 | — | — |
| 右侧 1 | 缩放 - | 🔍- | 缩小缩略图（最少 80px） | hover 时变 active | — |
| 右侧 2 | 缩放 + | 🔍+ | 放大缩略图（最大 320px） | hover 时变 active | — |
| 右侧 3 | **筛选** | ▽ | 打开 Filter By 弹出菜单 | 弹 modal | — |
| 右侧 4 | **排序** | ☰ | 切换排序方式（最近收藏/最早收藏/按时间） | 下拉 | — |
| 右侧 5 | **选择** | ☑ | 进入多选模式 | 进入选择模式 | — |

### 2.3 Filter By 菜单（Apple Photos 风格）

打开后弹出 360px 宽的菜单，支持多选：

```
┌──────────────────────────────┐
│ 筛选                        │
├──────────────────────────────┤
│ ❤  全部收藏                ✓ │
│ 🖼  仅照片                    │
│ ▶  仅视频                    │
│ ───────────                  │
│ ⏱  最近 7 天                 │
│ ⏱  最近 30 天                │
│ ⏱  最近一年                  │
└──────────────────────────────┘
```

| Filter 项 | 含义 | 实现 |
|-----------|------|------|
| 全部收藏 | 取消所有筛选 | — |
| 仅照片 | `is_favorite=1 AND file_type='photo'` | 后端 query param |
| 仅视频 | `is_favorite=1 AND file_type='video'` | 后端 query param |
| 最近 7 天 | `favorited_at >= now - 7d` | 后端 query param |
| 最近 30 天 | `favorited_at >= now - 30d` | 后端 query param |
| 最近一年 | `favorited_at >= now - 1y` | 后端 query param |

### 2.4 View Tabs

| Tab | 含义 | Active 状态 |
|-----|------|------------|
| 全部 | 显示所有收藏（含照片+视频） | 默认 |
| 照片 | 仅显示照片 | — |
| 视频 | 仅显示视频 | — |

### 2.5 PhotoCard 特殊操作

| 操作 | 触发 | 行为 | API |
|------|------|------|-----|
| 心形按钮 | 点击右上角 ❤ | toggle 收藏状态 | `POST/DELETE /api/photos/{id}/favorite` |
| 照片预览 | 双击 / Enter | 打开全屏预览 | — |
| 单选 | 普通点击 | 进入预览 | — |
| 多选 | 选择模式下点击 | toggle 选中 | — |
| 上下文菜单 | 右键 PhotoCard | 弹菜单 | — |

### 2.6 上下文菜单（右键 PhotoCard）

```
┌──────────────────────┐
│ 预览                │
│ 取消收藏 ❤           │
│ ────────             │
│ 加入相册...          │
│ ────────             │
│ 在资源管理器中显示   │
│ 导出...              │
│ ────────             │
│ 设为壁纸             │
│ 删除                │
└──────────────────────┘
```

| 菜单项 | 行为 | API |
|--------|------|-----|
| 预览 | 进入全屏预览 | — |
| 取消收藏 | `is_favorite=0` | `DELETE /api/photos/{id}/favorite` |
| 加入相册... | 弹"加入相册" modal | `POST /api/albums/{id}/photos` |
| 在资源管理器中显示 | 用 os 打开文件位置 | `subprocess.Popen(['explorer', '/select,', path])` |
| 导出... | 弹导出 dialog | — |
| 设为壁纸 | 设为系统壁纸 | Windows API |
| 删除 | 弹确认 → 删除文件+DB 记录 | `DELETE /api/photos/{id}` |

### 2.7 选择模式操作

进入选择模式后，顶部出现蓝色 banner：

```
[已选 3 张]                    [全选] [加入相册] [取消]
```

| Banner 按钮 | 行为 | API |
|-------------|------|-----|
| 取消 | 退出选择模式 | — |
| 全选 | 选中当前视图所有 | — |
| 取消全选 | 取消当前视图所有 | — |
| 加入相册 | 弹"加入相册" modal | `POST /api/albums/{id}/photos` |
| 删除 | 弹确认 → 批量删除 | `DELETE /api/photos?ids=1,2,3` |
| 导出 | 弹导出 dialog | — |

### 2.8 拖拽支持

| 拖动源 | 拖动目标 | 行为 |
|--------|----------|------|
| PhotoCard | Sidebar "相册集" 下的某相册 | 加入该相册 |
| PhotoCard | Sidebar "⭐ 我的收藏" | 加入收藏（多选模式下可批量） |
| Sidebar 相册 | 另一个相册 | 不允许（同级拖动） |

## 4. Sector 3：📅 按年份浏览

### 4.1 入口
- Sidebar: 年份 disclosure 列表（`2025 ▼ / 2024 ▶ / 2023 ▶`）
- 点击年份行 → 展开/收起月份列表（不切换视图）

- 工具栏标题：动态显示 `2024-10` + 数量 `203 张`

```
┌──────────────────────┐
│ 打开                │

│ 重命名...            │

| 右侧 3 | **排序** | 切换排序（按名称/按创建时间/按照片数） |

| Tab | 含义 | API |

### 5.6 Album Card 操作（主区域）

| 元素 | 操作 | 行为 |
|------|------|------|
| 缩略图 | 单击 | 进入该相册 |
| 缩略图 | **双击** | 进入该相册（与单击相同） |
| 名称 | **双击** | inline 编辑（input + Enter 确认） |
| 数量徽章 | hover | 显示 `点击查看 N 张照片` |
| 整张卡 | **右键** | 弹上下文菜单（同 §5.3） |

### 5.7 拖拽支持

| 拖动源 | 拖动目标 | 行为 |
|--------|----------|------|
| AlbumCard | sidebar 顶部 | 置顶 |
| AlbumCard | sidebar 底部 | 取消置顶 |
| AlbumCard | 另一个 AlbumCard | 排序（v0.7 不实现） |
| PhotoCard | AlbumCard（任何 sector） | 加入该相册 |

### 5.8 弹窗

- **新建相册 modal**：
  - 输入框（名称，最多 100 字）
  - 描述（可选，textarea）
  - 按钮：`取消` / `创建`
  - 提示："虚拟集合，不复制原文件"
  - API：`POST /api/albums {name, description}`
- **重命名 inline**：直接编辑名称 → Enter 确认
- **删除 modal**：见 §6.8

### 6.6 上下文菜单（右键 PhotoCard）

| 菜单项 | 行为 | API |

| 加入其他相册 | 弹"加入相册" modal | `POST /api/albums/{other_id}/photos` |
| 在资源管理器中显示 | 打开文件位置 | subprocess |
| 导出 | 导出 dialog | — |
| 删除 | 弹确认 → 删除文件 | `DELETE /api/photos/{id}`（同时从所有相册移除） |

### 6.7 选择模式

```
[已选 3 张]   [全选] [从相册移除] [加入收藏] [删除] [取消]
```

| Banner 按钮 | 行为 | API |
|-------------|------|-----|
| 取消 | 退出选择 | — |
| 全选 / 取消全选 | 切换全选 | — |
| **从相册移除** | 批量移除（不删照片） | `POST /api/albums/{id}/photos/batch-remove {photo_ids:[..]}` |
| 加入收藏 | 批量加入收藏 | `POST /api/photos/batch-favorite` |
| 加入其他相册 | 弹"加入其他相册" | `POST /api/albums/{other_id}/photos` |
| 删除 | 弹确认 → 批量删除 | `DELETE /api/photos?ids=1,2,3` |

### 6.8 删除相册 Modal

```
┌──────────────────────────────────┐
│  删除相册                    ×   │
├──────────────────────────────────┤

## 9. API 端点总览

### 9.1 收藏

| Method | Path | 说明 |
|--------|------|------|
| `POST` | `/api/photos/{id}/favorite` | 收藏 |
| `DELETE` | `/api/photos/{id}/favorite` | 取消收藏 |
| `POST` | `/api/photos/batch-favorite` | 批量收藏 `{photo_ids: [..]}` |
| `GET` | `/api/photos?favorite=true&page=N` | 列出收藏 |

### 9.2 相册

| Method | Path | 说明 |
|--------|------|------|
| `GET` | `/api/albums` | 列出所有相册（带 count） |
| `POST` | `/api/albums` | 创建相册 `{name, description}` |
| `PATCH` | `/api/albums/{id}` | 重命名/改描述/置顶 |
| `DELETE` | `/api/albums/{id}` | 删除相册（CASCADE album_photo） |
| `GET` | `/api/albums/{id}/photos?page=N` | 列出相册内照片 |
| `POST` | `/api/albums/{id}/photos` | 批量加入 `{photo_ids: [..]}` |
| `DELETE` | `/api/albums/{id}/photos/{photo_id}` | 单张移除 |
| `POST` | `/api/albums/{id}/photos/batch-remove` | 批量移除 `{photo_ids: [..]}` |

### 9.3 照片（已有 + v0.7 新增）

| Method | Path | 说明 |
|--------|------|------|
| `GET` | `/api/photos?path=2024-10&page=N` | 按目录列出 |
| `GET` | `/api/photos?favorite=true` | 列出收藏 |
| `GET` | `/api/photos?sort=date_desc&group_by=month` | 时间线分组 |
| `PATCH` | `/api/photos/{id}` | 修改 metadata（path, is_favorite 等） |
| `DELETE` | `/api/photos/{id}` | 删除照片 |
| `DELETE` | `/api/photos?ids=1,2,3` | 批量删除 |

### 9.4 文件夹（v1 + v0.7）

| Method | Path | 说明 |
|--------|------|------|
| `POST` | `/api/import/scan` | 增量扫描 `{path: xxx}` |
| `GET` | `/api/folders` | 列出物理文件夹（v0.7 暂未实现） |
